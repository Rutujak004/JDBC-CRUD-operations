 
# JDBC CRUD Operations

This project demonstrates basic **CRUD operations using JDBC in Java**.

## 💡 Features

- Create, Read, Update, and Delete operations using JDBC
- MySQL database integration
- Java-based console application

## 🛠️ Technologies Used

- Java
- JDBC
- MySQL
- Maven (if applicable)
